# Typescript Three.js Webpack Starter

- Typescript support.
- Webpack
- Html plugin

## Usage

### Install Typscript

```
npm install typescript -g
```
### Build Bundle

```
npm run build
```
*On progress*

### Start

```
$ git clone https://github.com/pinqy520/three-typescript-starter.git
$ cd three-typescript-starter
$ npm install # or yarn
$ npm start
```
